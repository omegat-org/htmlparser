Visitor

The HTMLRenderer interface is a visitor. It is used to visit a node and render it based on pluggable logic.

 		public interface HTMLRenderer {
  public String renderLinksToHTML(HTMLLinkTag linkTag);
  public String renderImagesToHTML(HTMLImageTag imageTag);
}

The HTMLNode class has the method toHTML() which takes in the visitor (renderer).

It defaults to calling the toHTML() method but is overridden by HTMLLinkTag and HTMLImageTag. Shown below is code from HTMLImageTag.

1
2
3
 		  public String toHTML(HTMLRenderer renderer) {
    return renderer.renderImagesToHTML(this);
  }
Java2html

HTMLLinkTag has a similar implementation :

1
2
3
 		  public String toHTML(HTMLRenderer renderer) {
    return renderer.renderLinksToHTML(this);
  }
Java2html

This mechanism allows clients to plugin logic for rendering links and images outside the parser. To see how this can be done, take a look at the Ripper example.
