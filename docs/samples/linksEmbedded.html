Extracting Embedded Links / Images

The architecture of the parser makes it very easy to extract embedded links or images from a page. All you need to do is to use the method node.collectInto()

Here's how you collect links from a page :

 		  HTMLParser parser = new HTMLParser("http://www.yahoo.com");
  parser.registerScanners();
  int i = 0;
  Vector collectionVector = new Vector();
  HTMLNode node;
  for (HTMLEnumeration e = parser.elements();e.hasMoreNodes();) {
    node = e.nextHTMLNode();
    node.collectInto(collectionVector,HTMLLinkTag.LINK_TAG_FILTER);
  }
  // All items in the collection vector should be links
  for (Enumeration e = collectionVector.elements();e.hasMoreElements();) {
    HTMLLinkTag linkTag = (HTMLLinkTag)e.nextElement();
    // you can now process the links as you like
  }

Line 8 is of particular importance here. Once you specify the collecting parameter, and the filter, the node matches itself against the filter. If the node contains a lot of nodes within it, then it can run through them passing your filter to each child node.

For extracting images, line 8 will change to
