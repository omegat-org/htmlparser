HTMLParser v 1.2

Welcome to the homepage of HTMLParser - a super-fast real-time parser for real-world HTML. What has attracted most users to HTMLParser has been its simplicity in design, speed and ability to handle streaming real-world html.

The production release of HTMLParser 1.2 will be out shortly (in a few weeks, hopefully before New Year's). The current integration release - Candidate 5 - is most likely to be designated as v1.2 production, if no bug reports come in.

You can download Candidate 5 (c5) from the download page.

Before you bother downloading the parser, you would probably want to check our "simple design" claim. Check sample programs to see how simple it is to parse HTML using HTMLParser.
