HTML Parser Mailing Lists

HTMLParser Announcement mailing list (very low traffic)
Join this list if you are interested in new releases of HTML Parser. Notifications of releases will be put on this list.

HTMLParser Developer mailing list.
Join this list ONLY if you plan to have a developer discussion about the htmlparser library. This list is intended for developers of HTMLParser only.

HTMLParser Users mailing list.
Join this list if you want to use the HTMLParser library and need some help to get started. Feel free to post your questions here.
